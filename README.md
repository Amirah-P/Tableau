# Tableau
<h1>Tableau - Seattle Airbnb Market Insights</h1>


<h2>Description</h2>
Project consists of a interactive dashboard in Tableau to analyse factors which impact average price and estimated revenue. Explored and incorporated geographic visualisations and time-series analysis to identify market patterns and regional differences, whilst building foundational skills in data visualisation and dashboard design. Originally following a tutorial which was then adapted with own ideas, creativity and needs.
